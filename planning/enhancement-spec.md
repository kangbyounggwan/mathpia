# Mathpia 앱 고도화 스펙

## 프로젝트 현황

Mathpia는 수학 학원의 선생님과 학생(중1~고3)을 연결하는 태블릿 전용 수학 학습 플랫폼입니다.

### 현재 구현된 기능
- **인증**: Mock 로그인 (teacher@test.com / student@test.com)
- **선생님 화면**: 대시보드, 학생관리, 숙제관리, 강의자료, 채점, AI 문제추출
- **학생 화면**: 대시보드, 숙제목록, 문제풀이(캔버스), 강의자료, 프로필
- **AI 문제 추출**: Gemini 2.5 Flash로 이미지/PDF에서 수학문제 추출 및 LaTeX 변환
- **수학 렌더링**: KaTeX/MathJax 기반 수식 표시
- **교육과정 데이터**: 중1~고3 전체 단원/챕터 체계

### 현재 기술 스택
- Expo React Native (SDK 54), TypeScript
- Zustand 상태관리
- React Native Paper UI
- Gemini AI API (@google/genai)
- expo-router 파일 기반 라우팅
- Drawing Canvas (필기 풀이)

### 미구현 사항
- 백엔드 연동 (Supabase 스키마는 설계 완료, 연동 미완)
- 실제 인증 시스템
- 문제은행 시스템
- 학습 분석/취약점 진단

---

## 고도화 요구사항

### 1. 문제은행(Question Bank) 시스템 구축

선생님이 관리하고 학생에게 출제할 수 있는 체계적인 문제 데이터베이스 시스템.

**핵심 기능:**
- 문제 등록: 수동 입력 + AI 자동 추출(기존 Gemini 서비스 활용)
- 문제 분류: 학년, 단원, 난이도, 유형(객관식/서술형/단답형)별 체계적 분류
- 문제 검색/필터: 다중 조건 검색 및 필터링
- 문제 태그: 자유 태그 시스템으로 유연한 분류
- 문제 통계: 사용 횟수, 정답률 등 메타데이터 관리
- 숙제 출제 연동: 문제은행에서 문제를 선택하여 숙제 구성

**참고:** DB 스키마의 `problem_bank` 테이블이 이미 설계되어 있음

### 2. 학생 취약 단원 분석 및 보완 학습 시스템

학생의 문제 풀이 데이터를 분석하여 부족한 영역을 진단하고, 맞춤형 보완 학습을 제공하는 시스템.

**핵심 기능:**
- **성취도 분석**: 단원별/유형별 정답률 추적 및 시각화
- **취약 단원 자동 진단**: AI 기반 또는 규칙 기반으로 부족한 영역 식별
- **맞춤형 문제 추천**: 취약 단원에서 적절한 난이도의 문제 자동 추천
- **학습 리포트**: 학생별/기간별 성취도 변화 추적
- **선생님 대시보드**: 반별/학생별 취약 단원 한눈에 파악
- **오답노트**: 틀린 문제 자동 수집 및 복습 기능

### 3. 추가 고도화 기능

수학 교육 앱으로서 경쟁력을 높이기 위한 추가 기능:

- **AI 풀이 도우미**: 학생이 문제를 풀다가 막힐 때 힌트/단계별 풀이 제공 (Gemini AI 활용)
- **실시간 알림**: 숙제 배정, 채점 완료, 마감 임박 등 실시간 알림
- **학습 게이미피케이션**: 연속 학습 스트릭, 뱃지, 레벨 시스템으로 동기 부여
- **Supabase 백엔드 연동**: 실제 인증, DB, Storage, Realtime 연동 완성
- **학부모 알림**: 학습 현황 요약을 학부모에게 공유

## 기술적 고려사항

- 기존 Gemini AI 서비스를 문제 추천 및 풀이 도우미에 확장 활용
- Supabase의 Realtime 기능으로 실시간 알림 구현
- 기존 curriculum.ts의 교육과정 체계와 연동
- Zustand 스토어 확장 (문제은행, 학습분석 등)
- 성능: 대량 문제 목록 가상화 렌더링, 분석 데이터 캐싱
